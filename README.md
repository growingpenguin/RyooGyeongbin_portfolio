>
> - 연구기간 : 2022.06.02-13
> - 핵심 역할 : 시뮬레이션을 통한 분석, 표준 개발, 특허 출원에 기여하고 있음. <br/>

> - 특허 출원: 경쟁사의 특허에 대해 주요 요소를 분석하고, 분류, 발명 명칭, 대표 도면, 요약, 대표 청구항 등을 검토하였습니다. 또한, 로봇친화적인 건물 환경에서 다양한 운영 및 제어 기술의 상호작용을 분석하고 시스템 최적화 방안을 연구하여 기업의 특허 출원에 기여하고 있음. <br/>
